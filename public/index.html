Music Visualizer via Processing

I built this music visualizer while learning to use Processing to create new media art. Processing is a tool that provides a great environment to make computational media such as images, animations, and interactions. I used the Minim audio library to detect beats in the song input in order to then create visualizations that correspond to the beats.

Prch

Prch is a cool new startup that aims to provide consumers with the information they need so that they can make more environmental and socially responsible purchasing decisions so that they can improve their quality of life. I interned at prch during my winter quarter in 2016 while attending UC San Diego. I developed the backend and javascript functionality as well as some of the frontend templates for the web application. It runs in a node environment and is currently hosted on Parse. Panjutsu

Panjutsu is a simple tool that helps UC San Diego students explore majors and career paths available to them. I worked with a team of 4 other UX designers and developers from ideation, sketching, storyboarding, prototyping to development as part of an information architecture and user experience course at UC San Diego. The main highlight of the tool was an intuitive and fun to use flowchart that showed the class path for a specific major. I designed and created the icons that shows which quarter a certain class is available. I also helped the team with scraping the information shown on the site as well as making improvements with the code. The whole process for creating this tool was recorded by my fellow teammember Amon which you can view here. Our app was voted the most helpful product from a class of about 40 students.
